# Sapere - v0.1.0

Sapere é uma plataforma de aprendizado focada em organizar conhecimento em trilhas estruturadas.

A ideia é permitir que alunos avancem por cursos, módulos e aulas, acompanhando sua evolução ao longo do processo de aprendizagem.

## Stack Tecnológica

Backend

* Java 21
* Spring Boot
* Spring Data JPA
* PostgreSQL

Ambiente de Desenvolvimento

* Maven
* Lombok
* H2 Database

## Arquitetura

O projeto segue uma arquitetura em camadas:

Controller
Service
Repository
Domain

Conceitos principais do domínio:

Usuário
Curso
Módulo
Aula
Matrícula
Progresso
